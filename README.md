## ⚡ Hakkımda

Yazılım mühendisliği öğrencisi olarak teknolojinin hem savunma hem de analiz tarafına odaklanıyorum. Karmaşık sistemlerin en alt seviyedeki çalışma mantığını çözmek ve dijital izleri takip etmek ana tutkum.

* 🔭 **Şu An:** Malware analizi ve adli bilişim (Digital Forensics) süreçleri üzerine yoğunlaşıyorum.
* 🎯 **İlgi Alanları:** Tersine Mühendislik, Malware Analizi ve Tehdit Avcılığı.
* 🌱 **Öğrenim Yolculuğu:** Web teknolojilerine hakimim; ancak odağımı tamamen siber güvenlik ve sistem güvenliği mimarilerine kaydırdım.
* 💬 **Soru Sorun:** Tersine Mühendislik, Adli Bilişim ve C#/.NET güvenliği hakkında konuşmayı severim.
